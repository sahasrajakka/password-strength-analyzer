# Password Strength Analyzer

A Python-based tool that evaluates password strength based on length, complexity, and uniqueness. The application provides feedback, suggests stronger passwords, and prevents password reuse using secure password hashing.

## Features
- Password strength analysis
- Length and complexity checks
- Common password detection
- Strong password suggestions
- SHA-256 password hashing
- Password reuse prevention with SQLite

## Technologies Used
- Python
- SQLite
- Hashlib (SHA-256)
- Regular Expressions

## How to Run

```bash
python password_analyzer.py
